## PiP / Kompaktmodus

| Option | Description | Selected |
|--------|-------------|----------|
| Im Tab, fixiert | Bleibt im Browser-Tab, kein API | |
| Echter Browser-PiP | Document PiP API, Chrome 116+ | ✓ |

**User's choice:** Document Picture-in-Picture API (`window.documentPictureInPicture`), NOT Canvas-Trick. Draggable + resizable (browser-native). Position/Größe merken via localStorage (nice-to-have).

**Kompakt-Panel Spec (user-defined):**
- Mini-Header: Modus + Timer
- EIN KI-Tipp (groß, lesbar) — KI steuert automatisch
- Kleine Tabs oben für User-Override
- 2 EWB-Vorschläge (KI wählt) + "Alle ▾"
- Link zu Vollbild
- NICHT im Kompakt: Sprachanalyse, Kaufbereitschaft %, Phasen-Leiste, volles Transkript

## Dashboard Layout
