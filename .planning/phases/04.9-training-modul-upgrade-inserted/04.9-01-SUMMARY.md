# Phase 04.9 Plan 01: DB Foundation — PersonalityType Model + System Seeds Summary

PersonalityType SQLAlchemy model with 6 system personality types seeded, ConversationLog extended with personality_type_id and stimmung_history, difficulty labels renamed to Einsteiger/Fortgeschritten/Experte, and 7 DACH-B2B system training scenarios added.

## What Was Built

### Task 1: PersonalityType model + ConversationLog extension + migration

**database/models.py:**
- Added `PersonalityType` class (10 columns: id, user_id, org_id, is_custom, name, icon, kurzbeschreibung, attribute, kommentar, erstellt_am)
- Extended `ConversationLog` with `personality_type_id` (FK to personality_types) and `stimmung_history` (Text/JSON)

**app.py `_migrate()`:**
- `CREATE TABLE IF NOT EXISTS personality_types` — all columns matching the model
- `ALTER TABLE conversation_logs ADD COLUMN personality_type_id INTEGER` (try-except)
- `ALTER TABLE conversation_logs ADD COLUMN stimmung_history TEXT` (try-except)
- `INSERT OR IGNORE INTO personality_types` for 6 system types with full `attribute` JSON (geduld, skeptik, zeitdruck, startstimmung, auflege_trigger_hart/weich, beispiel_reaktionen, verhaltensregeln, position_profil)

**app.py `_seed_system_training_scenarios()`:**
- New seed function for 7 DACH-B2B system scenarios with `erstellt_von=NULL` as system marker
- Scenarios cover: IT/SaaS Cloud-Migration, Maschinenbau Automatisierung, Versicherung bAV, Immobilien Gewerbeflaechen, IT-Security Audit, SaaS CRM-Wechsel, Maschinenbau Wartungsvertrag
- Called in startup sequence after `_seed_training_scenarios()`

### Task 2: SCHWIERIGKEITEN labels + PERSONALITY_TYPES_SEED constant

**services/training_service.py:**
- `SCHWIERIGKEITEN['leicht']['label']` updated to `'Einsteiger'`
- `SCHWIERIGKEITEN['mittel']['label']` updated to `'Fortgeschritten'`
- `SCHWIERIGKEITEN['schwer']['label']` updated to `'Experte'`
- `SCHWIERIGKEITEN['sekretaerin']['label']` unchanged (`'Sekretärin + Chef'`)
- Added `PERSONALITY_TYPES_SEED` list constant with 6 entries (name, icon, kurzbeschreibung, startstimmung, geduld, skeptik, zeitdruck)
