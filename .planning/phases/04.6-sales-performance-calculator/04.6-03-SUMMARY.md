# Phase 04.6 Plan 03: CSS + JavaScript — Performance Calculator Summary

Chart.js charts, debounced slider simulation, API fetch wiring, Deal-Wert modal logic, and Ergebnis-Tagging all implemented as a single self-contained script block in dashboard.html backed by teal-thumb slider CSS in nerve.css.

## What Was Built

**Task 1 — Slider CSS in nerve.css**

- `/* Sales Performance Calculator */` section appended to `static/nerve.css`
- `input[type="range"].perf-slider` CSS: `-webkit-appearance:none`, height 6px, `#E5E7EB` track, border-radius 3px
- `::-webkit-slider-thumb` and `::-moz-range-thumb` both styled teal `#00D4AA`, 18px circle, teal box-shadow
- `#perf-kpis` responsive grid: 4 columns → 2 at 900px → 1 at 480px (display:none by default, JS switches to grid)
- `#perf-charts` responsive grid: 2 columns → 1 at 768px (display:none by default)
- `.perf-slider-group` bottom margin 4px; mobile media query for sim-panel slider grid
- `class="perf-slider"` added to all 5 range inputs: `sl-calls-tag`, `sl-show-rate`, `sl-closing-rate`, `sl-deal-wert`, `sl-arbeitstage`

**Task 2 — Performance JavaScript in dashboard.html**

- `<script src="https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js">` added (Chart.js was not in dashboard.html or base.html)
- `formatEuro()`: rounds to full euros, handles Mio/K suffixes with German locale
- `formatEuroShort()`: compact K/Mio display for KPI cards and slider labels
- `formatPercent()`: configurable decimal places, German comma separator
- `perfSetMode(mode)`: toggles real/sim button styles, shows/hides `perf-sim-panel`, calls `perfRenderReal()` or `perfSimUpdate()`
- `perfRenderReal(d)`: fills `kpi-deals`, `kpi-umsatz`, `kpi-forecast`, `kpi-roi`; hides loading/empty; shows/hides KPI+chart grids; pre-fills slider defaults from real data; auto-switches to sim mode when `hat_daten=false`
- `perfRenderBarChart(chartData)`: Chart.js bar chart on `call-activity-chart` canvas — two datasets (Calls teal, Abschluesse darker teal), destroy-recreate pattern
- `perfRenderGrowthChart(d)`: Chart.js line chart on `growth-chart` canvas — 12-month S-curve forecast at 5% monthly growth from `umsatz_pro_monat`, teal fill, `formatEuroShort` tooltip
- `perfSimUpdate()`: reads all 5 slider values, updates labels immediately, debounced 150ms calculation: calls/monat × show-rate × closing-rate × deal-wert → umsatz; 12-month S-curve sum; ROI with 15% Einwand-Verbesserung assumption - 99€ Abo
- `openDealWertModal()` / `closeDealWertModal()`: show/hide fixed overlay, auto-focus input
- `saveDealWert()`: validates input ≥1, PATCH `/api/user/deal-wert`, closes modal, pre-fills sl-deal-wert, calls `perfInit()`
- Enter-key listener on `deal-wert-input` via DOMContentLoaded
- `tagSession(result)`: guards on `_tagSessionId`, PATCH `/api/session/{id}/result`, highlights active button (teal active, grey others), hides bar after 2s
- `perfInit()`: reads `?tag_session=` URL param → shows `result-tag-bar`, cleans URL via `history.replaceState`; fetches `/api/performance`; shows deal-wert-modal if `avg_deal_wert` null; calls `perfRenderReal()` + `perfSimUpdate()`
- DOMContentLoaded / immediate start guard for `perfInit()`
