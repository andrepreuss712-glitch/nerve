| Option | Description | Selected |

## Zusätzliche UX-Fixes (vom User eingebracht)

1. **Profil-Popup:** Modal/Pop-up wenn "Anrufen" ohne Profil geklickt wird — aktuell nur blasser Hinweistext
2. **Setup-Flow linear:** Kompletter Umbau — alles sichtbar, keine Hide/Show-Mechanik. Reihenfolge: Verkaufsprofil → Schwierigkeit → Kundentyp → Anruf-Typ → Szenario → Modus → Anrufen
3. **Szenario-Karten Design:** Ans NERVE Glass-Design anpassen (wie Schwierigkeits-Karten)
