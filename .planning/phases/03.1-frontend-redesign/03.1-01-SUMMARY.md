# Phase 03.1 Plan 01: CSS Foundation + Base Layout Summary

NERVE Design System CSS delivered as `static/nerve.css` with all custom properties, component classes, and layout helpers; `base.html` updated to link nerve.css, load Lucide, and use `.n-nav-item` / `.n-nav-item-active` classes — every child page now inherits the new dark-glass design language automatically.

## What Was Built

### Task 1: static/nerve.css (684 lines)

Created the single-source CSS file for the entire NERVE application UI:

- **Google Fonts import** — Inter (300-800) + JetBrains Mono (400-600)
- **:root custom properties** — 19+ variable groups covering page/body, glass panel, all button variants (primary/accent/ghost/danger), inputs, labels, KPI cards, badges, nav items, session rows, metric bars, pill tabs, AI panel, live badge, feed items, avatars, hero cards, promo cards
- **All .n-* component classes** — 12+ component families: `.n-glass`, `.n-btn-*`, `.n-input`, `.n-label`, `.n-kpi`, `.n-badge-*`, `.n-nav-item`, `.n-session`, `.n-metric`, `.n-pill`, `.n-ai-panel`, `.n-live-badge`, `.n-feed-item`, `.n-avatar`, `.n-hero`, `.n-promo`, text helpers, `.lucide` sizing
- **Layout helpers** — `.g-header`, `.g-body`, `.g-sidebar`, `.g-content` (replaces base.html inline layout CSS)
- **Legacy class preservation** — `.flash`, `.btn`, `.badge`, `.card`, `.form-*`, `.tbl`, `.nav-mark`, etc. so unmigrated child templates continue working without breakage
- **Brand update** — Nav logo N border + color changed from gold `#E8B040` to NERVE teal `#2dd4a8`

### Task 2: templates/base.html

Replaced 178-line inline `<style>` block with nerve.css link:

- **Removed** Playfair Display + DM Sans Google Fonts link (now via `@import` in nerve.css)
- **Removed** entire inline `<style>` block
- **Added** `<link rel="stylesheet" href="{{ url_for('static', filename='nerve.css') }}">`
- **Added** `<script src="https://unpkg.com/lucide@latest"></script>` in `<head>`
- **Updated** all 10 sidebar nav items: `.nav-item` / `.active` → `.n-nav-item` / `.n-nav-item-active`
- **Updated** logout button: `.g-logout` → `.n-btn .n-btn-ghost .n-btn-sm`
- **Added** `<script>lucide.createIcons();</script>` before `{% block scripts %}`
- **Preserved** all Jinja2 blocks, changelog popup, auto-logout warning, cancel banner, help FAB
