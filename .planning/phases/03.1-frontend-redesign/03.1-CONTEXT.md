<decisions>
## Implementation Decisions

### Architecture

- **Locked: CSS delivery** — Single `static/nerve.css` file. All CSS Custom Properties (`:root {}`) and component classes (`.n-glass`, `.n-btn-primary`, etc.) live here. No inline `<style>` blocks in app pages after redesign.
- **Locked: base.html** — Updated first. Links `nerve.css`, replaces header/sidebar/nav/layout CSS with NERVE classes. All child templates inherit this automatically.
- **Locked: plan split** — Foundation first: Plan 1 = nerve.css + base.html. Plans 2–N = individual app pages built on top of the stable foundation.
- **Locked: fonts** — Switch from `Playfair Display` + `DM Sans` to `Inter` (300–800) + `JetBrains Mono` (400–600). Google Fonts import in nerve.css.
- **Locked: Lucide icons** — Design system uses `<script src="https://unpkg.com/lucide@latest">` — add to base.html head if not already present.
- **Locked: no framework** — Pure CSS with Custom Properties. No Tailwind, no React, no CSS-in-JS.
