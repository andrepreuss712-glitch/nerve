# Phase 03.1 Plan 04: Training Page NERVE Design System Migration Summary

Migrated training.html from hardcoded color palette to NERVE design tokens — glass panels for difficulty/mode cards, teal primary accents for call/PTT buttons and timer, KI-gold for AI speaker elements, all three training phases updated consistently.

## What Was Built

**Task 1: Migrate training.html inline CSS to NERVE design tokens**

Complete color token migration across all four training phases (setup, call screen, chat, scoring):

- **Background/surface tokens**: `#0c0c18` → `var(--page-bg)`, `#1a1a2e` → `var(--glass-bg)`, border colors → `var(--glass-border)` / `var(--glass-border-hover)`
- **Text tokens**: `#f0ebe0` → `var(--page-text-color)`, `#6b6b80` → `var(--page-text-muted)`, secondary text → `var(--page-text-secondary)`
- **Primary accent**: `#E8B040` gold → `#2dd4a8` teal for call button, PTT button, timer, language selector active state, text input focus border, send button text, sys messages
- **KI/AI accent**: `#d4a853` for avatar initials, `.t-msg-name`, bubble-out border and background, help suggestion colors — semantic distinction between user-action teal and AI-speaker gold
- **Glass cards**: `.t-diff-card` and `.t-modus-card` updated to `var(--glass-bg)` background with `var(--glass-border)` borders; active state uses teal border + `rgba(45,212,168,0.06)` tint
- **n-glass class**: Added `n-glass` to difficulty card divs in the HTML template
- **Score SVG**: Bar fill colors updated — ≥70 → `#2dd4a8`, ≥50 → `#fbbf24`, <50 → `#e05c5c`; circle text font updated from `'DM Sans'` to `'JetBrains Mono'`
- **Live preview section**: Updated to use teal border/badge styling; CTA button updated to teal
- **font-family overrides removed**: `.t-call-btn`, `.t-help-btn`, `.t-send-btn` no longer specify `font-family:inherit` (inherits Inter from nerve.css body)

**Preserved exactly:**
- All Jinja2 template blocks and loops (`{% for p in profiles %}`, `{% for key, diff in schwierigkeiten.items() %}`, etc.)
- All JavaScript functions (scenario selection, difficulty/mode selection, chat, voice recording, timer, help suggestions, scoring, freizeichen tones)
- All element IDs used by JavaScript
- All data-* attributes
- `.t-overlay` / `.t-hidden` phase transition mechanism
- `@keyframes pulse-ring` (call screen animation)
- `@keyframes pulse-rec` (recording animation)
- `@keyframes msg-in` (chat message animation)
- ElevenLabs voice playback logic
- Training mode selection (frei/gefuehrt)
- Red recording state (`.t-ptt-btn.t-recording { background:#e05c5c }`)
