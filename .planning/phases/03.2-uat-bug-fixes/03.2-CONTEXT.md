### 2. Compact Mode Implementation (UX-09)

**Decision:** CSS overlay inside the same window (NOT a new browser popup window).

**Rationale:** A new window would break the Socket.IO connection. The existing `toggleKompakt()` adds a `.kompakt` class to `body` which hides the main content and shows a compact panel — that's already close to an overlay. The fix is to style the compact panel as a `position:fixed` floating overlay on top of everything (like `frage-overlay`), so it can be dragged or positioned over the screen without changing the window.

**Behavior:**
- Compact panel opens as `position:fixed` overlay (bottom-right corner, draggable optional)
- Main content stays visible underneath (semi-transparent)
- "◀ Kompakt" button opens overlay; "▶ Vollbild" or ✕ closes it
- Socket stays connected; all real-time data still flows
