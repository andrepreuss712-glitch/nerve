## Decisions Made

- Timer guard in transcript handler kept as `if (!sessionTimer)` fallback — covers race condition where `connect` fires before DOM is ready
- No-conversation guard for training checks `.t-msg-out` class count — outgoing messages use this class, reliable proxy for "did user actually speak"
