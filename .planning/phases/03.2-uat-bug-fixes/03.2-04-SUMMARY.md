## Decisions Made

- No-FOUC script placed before CSS `<link>` — runs synchronously before any render, prevents flash of wrong theme
- Server hint via `data-server-theme` takes precedence on page load — ensures DB preference wins over stale localStorage
- `preferred_theme` defaults to `'dark'` — fully backward-compatible, no visual regression for existing users
- Sun icon shown in dark mode (clicking → light), moon shown in light mode (clicking → dark) — next-state indicator pattern
- `toggleTheme()` fires DB sync via `fetch()` with `.catch(function(){})` — silent failure if unauthenticated (e.g., on public pages that still extend base.html)
