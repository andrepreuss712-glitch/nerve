  <files>static/nerve.css</files>
  <action>
In static/nerve.css, make two changes:

1. On `.g-sidebar` (line ~549-550): remove `overflow-y:auto` and `gap:2px` from the rule. The rule becomes:
```css
.g-sidebar{width:204px;background:rgba(255,255,255,0.02);border-right:1px solid rgba(255,255,255,0.06);
  display:flex;flex-direction:column;flex-shrink:0;padding:8px;}
```

2. After the `.g-sidebar::-webkit-scrollbar-thumb` rule (line ~552), add the new inner div rule:
```css
.g-sidebar-inner{flex:1;overflow-y:auto;min-height:0;display:flex;flex-direction:column;gap:2px;}
.g-sidebar-inner::-webkit-scrollbar{width:3px}
.g-sidebar-inner::-webkit-scrollbar-thumb{background:rgba(255,255,255,0.06)}
```

This makes .g-sidebar-inner take all available vertical space (flex:1), scroll its contents, and the sidebar-user-block (which has margin-top:auto and flex-shrink:0) will stay pinned at the bottom of .g-sidebar because it is a flex sibling, not inside the scrollable inner div.

Note: .sidebar-user-block already has `margin-top:auto` and `flex-shrink:0` in the existing CSS — no changes needed there.
